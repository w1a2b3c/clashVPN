ClashForWindows是什么？

ClashForWindows是一款基于Clash的Windows GUI代理客户端。关于它的安全性优势，以下几点可能会对你有所帮助：

隐私保护：作为一款代理客户端，ClashForWindows可以帮助用户在浏览网页时隐藏自己的IP地址，保护用户的在线隐私。
加密通信：ClashForWindows支持多种加密方式，包括但不限于Shadowsocks、VMess、Trojan等，可以保护用户的网络通信不被窃听。
规则分流：ClashForWindows具有强大的规则分流功能，用户可以根据自己的需求设置不同的代理规则，防止不必要的数据泄露。
透明代理：通过ClashForWindows，用户可以实现全局或者按需代理，有效避免了部分网站或应用对IP地址的限制。
开源：ClashForWindows是开源软件，其源代码可以公开查阅，这意味着任何人都可以查看其内部实现，从而确保没有隐藏的恶意代码。

ClashForWindows界面功能说明
Clash主界面
General（常规）：
Port：设置 HTTP、SOCKS5 代理端口，其他应用程序可配置该代理端口实现科学上网；
Allow LAN：启用局域网共享代理功能，开启后局域网的其他机器可使用ip:port代理实现科学上网；
Log Level：日志等级，info表示常规信息级别、debug表示显示详细调试信息；
Home Directory：点击后进入clash用户配置文件夹，该文件夹主要包含clash订阅配置；
GeoIP Database：点击下方日期可更新 GeoIP 数据库，存储世界国家IP地址库，文件较大，更新可能较慢；
Service Mode ：将Clash设置为系统服务；
Tun Mode：一种特殊的网络模式，它利用虚拟网络设备TUN实现了全系统的代理；
Mixin：高级功能，一种配置混合和快速切换的策略，让你可以快速地在不同的配置文件间切换，同时也可以在不同配置文件之间混合使用各种设置；
System Proxy：启用系统代理，开启后电脑常用浏览器会开启代理，实现科学上网；
Start with Windows：设置开机自动启动；
Proxies（代理）：选择代理方式（Global – 全局、Rule – 规则、Direct – 直连）及策略组节点选择；
Profiles（配置管理）：
用来下载远端配置文件和创建本地副本，且可在多个配置文件间切换；
对配置进行节点、策略组和规则的管理（添加节点、策略组和规则在各自编辑界面选择 Add, 调整策略组顺序、节点顺序及策略组节点使用拖拽的方式）；
Connections (连接): 显示当前的 TCP 连接，可对某个具体连接执行关闭操作；
Settings（设置）：软件详细设置；

选择代理级别

在软件左侧点击Proxies代理菜单，选择相应的代理级别，常规一般是从Global和Rule中选择。

代理级别区别：

Global：全局模式。在这个模式下，所有的网络请求都会通过代理服务器发送，而不论这些请求的目的地是什么。这对于那些需要对所有流量都进行代理的用户来说非常有用。

Rule：规则模式。在这个模式下，网络请求是否通过代理，取决于设置的规则，也就是从机场导出的配置提供的默认代理规则。用户可以设置一些规则来决定哪些流量需要走代理，哪些流量直连。这对于那些只需要对特定流量进行代理的用户来说非常有用。

代理级别新手设置建议：

直接选择Rule模式，这样不会影响本机其他的应用程序。

打开系统代理及开机自启选项

返回 General常规 中，打开System Proxy(系统代理)
